### Prerequisites

For this section please copy **100GB** of tpc-ds data to your S3 bucket from **s3://redshift-downloads/TPC-DS/100GB/**

```
e.g.

aws s3 cp --recursive s3://redshift-downloads/TPC-DS/100GB/ s3://analytics-on-aws-demo-100gb/100GB/
```
